# Core sections 

This folder represents core sections of the Kaspi Shop webpage

1. [Header](./header) section represents main header and navigation of the Kaspi web application platform
2. [Navigation] section represents the actual Kaspi Shop Application header that includes shop button, search bar, and city filter.
3. [Footer](./footer) section represents additional Kaspi information and platform links.

Each section follows Module based approach and has its own components, interfaces and styles that are not reusable in other section. All reusable part go to the [Shared](../shared) folder.
